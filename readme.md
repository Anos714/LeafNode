# 🌿 LeafNode – Notes That Grow With You

**LeafNode** is a modern **MERN-stack notes application** inspired by nature and simplicity.  
It helps you capture, organize, and grow your ideas in a clean, calm, and productive environment.

> 🚧 **Status:** Work in Progress  
> This project is actively being developed. Features and structure may change.

---

## ✨ Vision

LeafNode aims to be:
- 🌱 Simple yet powerful
- 🌿 Calm, distraction-free, and nature-inspired
- ⚡ Fast, secure, and developer-friendly

A place where your thoughts can **grow naturally**.

---

## 🛠️ Tech Stack (Planned)

### Frontend
- React.js (Vite)
- Redux Toolkit
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB & Mongoose
- JWT Authentication

### Deployment
- Vercel (frontend)
- Render (backned)
